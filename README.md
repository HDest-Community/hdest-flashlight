# HDest Flashlight
_Originally "DarkDoomZ (Ace Bootleg)", made by Accensus, now maintained by the communtiy._  
_**NOTE: No longer includes DDZ, feel free to use whatever relighting mod you wish.**_  

Small mod that adds a Flashlight item that requires Cell Batteries to operate (press Use + Use Item to pull up the flashlight).  The flashlight will start flickering when your battery is low, so make sure to keep them stocked up!

## Notes
- Loadout Code: `ddf`
- The flashlight can only spawn in backpacks.
- To bring the flashlight up, press Use + Use Item.
